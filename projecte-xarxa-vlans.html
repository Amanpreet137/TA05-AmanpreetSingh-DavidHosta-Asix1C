Xarxa amb VLANs, STP i router-on-a-stick

Disseny i configuració d’una xarxa d’oficina segmentada per departaments, evitant bucles amb STP i permetent comunicació inter-VLAN a través d’un router-on-a-stick.

Tornar a l’inici Veure tots els projectes

Fitxa tècnica

Rol: Disseny i configuració

Entorn: Cisco Packet Tracer / laboratori

Àrees: VLAN, trunking, STP, routing

Equipament: Switchos L2 + Router (ROAS)

Detalls del projecte

Objectius

  • Separar trànsit per departaments amb VLANs
  • Configurar enllaços troncals 802.1Q
  • Evitar bucles amb STP i definir root bridge
  • Permetre interconnexió entre VLANs (router-on-a-stick)

Topologia (inventada però realista)

  • VLAN 10: Administració
  • VLAN 20: IT
  • VLAN 30: Direcció
  • VLAN 99: Gestió (nativa / management)
  • Switch Core + Switch Accés + Router

Implementació (resum)

  • Creació de VLANs i assignació de ports d’accés
  • Configuració de trunks entre switchos (allowed VLANs)
  • STP: prioritat per elegir root bridge i verificació
  • Subinterfícies al router: encapsulation dot1q + IP gateway
  • Proves: ping inter-VLAN i comprovacions de taules

Resultats

La segmentació redueix broadcast entre departaments i millora la seguretat. STP garanteix estabilitat davant bucles i el router-on-a-stick proporciona connectivitat inter-VLAN amb un únic enllaç tronc.
